=====

1. Reprezentarea starilor
```
(m, 1, 1, 1, 1)
```
2. Identificare stari speciale
`SI`, `SF`, `(m, 3, 3, 3)`
3. Validare de tranzitii
4. Strategii
  * Random
  * Hill Climbing
  * Backtracking
  * `A*`

```C
strategie_BKT(sc) {
	for i = 1 .. n
		for j = 1 .. m
			SN = tranzitie(SC, i, j)
			if valid(SN)
				SC = SN
				strategie_BKT(SC)
}
```


```C
strategie_HC() {
	while (!st_finala(SC)) {
		i = random(1..n)
		j = random(1..m)
		SN = tranzitie(SC, i, j)
		if (valid(SN)) {
			if (scor(SN) >= scor(SC)) {
				SC = SN
			}
		}
	}
}
```
